Constraints in SQLSERVER
=========================
constraints are apply on table columns
Primary Key(does not allow duplicates and table can have only one primary key)
Foreign Key(refer to primary key value of other table,make relation between tables)
Unique Key(does not allow duplicates and allow one null value,can have multiple unique key columns in a table)
Not null(can not allow null values)
Check(check the value while inserting or updating)(apply validations on the column)
default(set default value to the column)
Note: default constraint is null
